# Chladni_Cymatics_Studio
Cymatics studio modelled based upon Chladni's resonance theory of Chladni plate's, but unlike one resonant pattern a plate of set dimension gives this software emulates a universal virtual world of particles 
meaning the higher you set the settings the harder it is to create vivid patterns which means it can be used for music practices on piano etc

If you set audio auto-morph and phase breathing speed too high, it becomes as hard to create vivid patterns as driving a high torque vehicle on an icy road, which means
it is amazing for training your piano skills as clarity of playing becomes immediately visible bcause only the most harmonic music will create vivid patterns with high settings

Mode A (Low) and Mode B (High) shouldn't be set to same value, otherwise the picture becomes static
if you set both Mode A and mode B with two values that are let's say 2 apart,
it creates a diagonal line through whole screen as you can see in the screenshot where
mode A is 7 - 9 and mode B is 10 - 12

If you set plate trapping and sand shimmer too high, the sand becomes as responsive as marble balls
on a polished floor. If you set them high, it shows you clarity of your playing
because well distinguised shapes and lines are almost impossible if the
melodies are dissonant, meaning setting every value to high works only for
the most harmonic music, no matter how complex the music gets

Particle count is highly optimised and works well even on low-end PC that can easily simulate 30 thousand particles in real time
Particle count is highly dependent on Mode A and mode B values, so if you set them high to let's say
A (6 - 8) B (9 - 12) the particle count should be around 40 thousand or higher for the best clarity.
